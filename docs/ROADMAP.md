
Roadmap
Phase 1 - Foundation

Create the repository foundation, documentation, governance boundaries, and project structure.

Phase 2 - RAG Layer

Build document loading, chunking, retrieval, and citation metadata.

Phase 3 - Agent Layer

Create the response generation interface and agent workflow.

Phase 4 - Evaluation Layer

Implement scoring for groundedness, relevance, citation coverage, hallucination risk, and policy compliance.

Phase 5 - Governance Evidence Layer

Generate durable evidence packages for each evaluated run.

Phase 6 - Dashboard

Create a reviewer-facing interface for evaluation history, evidence review, and approval workflow.

Phase 7 - Azure Integration

Add Azure AI Search, Azure OpenAI, and Prompt Flow integration.
